# Chaînage Efficace de Contour

This repository contains the archived source code of Chaînage Efficace de Contour, a code developed between 1985 and 1991 at Inria on the topic of contour chaining in the field of computer vision. Autors are Gérard Giraudon (Inria), Philippe Garnesson (Inria) and Patrick Cipière (Inria).

The most comprehensive description can be found in the Inria research report "Chaînage efficace de contour", No. 605, February 1987, by Gerard Giraudon:
https://inria.hal.science/inria-00075949/document

The work described in this report has led to several international publications:

"An Efficient Chain-Linking Algorithm", G. Giraudon, in The 5th Scandinavian Conference on Image Analysis, Stockholm, June 1987.
"A Real Time Parallel Edge Following in Single Pass", G. Giraudon, in Workshop on Computer Vision, Miami, 1987.
"Chaînage efficace de contour", G. Giraudon, 3ème Colloque Image-Cesta, Paris, Mai 1987.
Additional source code complements this work and is documented in research and technical reports included in the archive:

"Un standard pour une représentation d’objets de type liste dans le domaine du traitement d’images", No. 82, December 1988 – 3rd edition, P. Garnesson, G. Giraudon.
https://inria.hal.science/inria-00070061v1

"L’approximation polygonale, bilans et perspectives", INRIA Research Report No. 1621, June 1991, P. Garnesson & G. Giraudon.
https://inria.hal.science/inria-00074940v1/document

This work has been cited and led to a hardware implementation presented at the 16ème COLLOQUE GRETSI — 15-19 SEPTEMBRE 1997:

https://www.semanticscholar.org/paper/Chainage-efficace-de-contour-Giraudon/c357c726ab5a5bb0a9e641bc490fa8583769159f
https://www.gretsi.fr/data/colloque/pdf/1997_001-0287_12820.pdf
https://www.researchgate.net/publication/27608096_Operateur_materiel_de_chainage_de_contours_temps_reel_et_flot_de_donnees


## Curation

Material was retrieved and submitted by Gérard Giraudon. Archival was done by Mathilde Fichen (Software Heritage)


## Main Branch

The main branch contains the original materials as well as the metada linked to the code of Chaînage de contours. 
The original finds are stored in the Depository containing the raw materials.
- Folder [raw materials](./raw_materials) is for the original source code materials, as they have been found or submitted.
- Folder [source_code](./source_code) is for a machine readable version of the source code. 
- Folder [metadata](/.metadata) holds various files with meta information to be updated throughout the process. 


## SourceCode Branch
The SourceCode branch contains the code as it would have been see historically.
